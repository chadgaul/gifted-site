## SEO Keyword Strategy (researched 2026-07-15)

### Positioning insight

Competitors (Elfster, Giftster, GiftList) own "shared wishlist" and "secret santa" - social/group gifting. Gifted's differentiation is the **private, personal gift command center**: budgets per person + per event, gift status pipeline, reminders, privacy/iCloud. Target keywords match that intent - searchers planning and tracking their own gift-giving, not sharing lists. These searchers convert because the app is the direct answer to their problem.

### Primary keywords (landing page)

| Keyword | Intent | Where used |
|---|---|---|
| christmas list app | head term, matches app name | title tag, H1, hero |
| gift tracker app / gift tracking app | evergreen, year-round | title tag, hero, features |
| christmas gift list tracker | high intent, seasonal peak Oct–Dec | meta description, features |
| gift budget tracker | budget-focused, matches subtitle | features, screenshots section |

### On-page SEO checklist applied to the site

- Unique title (< 60 chars) + meta description (< 155 chars) per page
- One H1 per page containing the target keyword
- JSON-LD: SoftwareApplication (home), FAQPage (home), HowTo/Article + BreadcrumbList (guides)
- Internal linking: FAQ "Learn More" → guides; each guide cross-links 2–3 sibling guides + home
- Canonical URLs, Open Graph + Twitter cards, descriptive alt text on all screenshots
- sitemap.xml + robots.txt
- Real content depth on guides (800+ words), answering the query before pitching the app
